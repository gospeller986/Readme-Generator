# Readme-Generator

This is a hobby project created to offload the boring task of writing ReadMe to a LLM which can do it much better than us . 

## TechStack 
- Next js
- Express js
- Github Apis 
- Open Ai Apis

## Steps to install 

### SERVER

- clone the repo 
- Run the command `npm i `
- Run the command `npm run start`

### CLIENT 

- Run the command `npm i `
- Run the command `npm run dev`
